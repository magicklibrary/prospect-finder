Frequently Asked Questions

Does this cost anything?
No. The tool is a free HTML file. ChatGPT has a free tier that covers all searches. There is no API key, no subscription, and no usage fee.

How accurate is the data?
ChatGPT searches the web in real time for each query, so results reflect publicly available information. Company names, industries, and websites are generally accurate. Phone numbers and contact names should always be verified before calling â€” treat them as a starting point, not a guaranteed source of truth.

Can I search the same state twice?
Yes. Each search is independent and may return different companies. ChatGPT introduces natural variety, so running the same state twice often yields a different set of prospects.

What if the Load Results button doesn't work?
The tool is looking for a block of structured data (JSON) in ChatGPT's response. If it fails: make sure you copied the entire response from top to bottom, including the code block at the end. If ChatGPT only returned text without a code block, ask it: "Please format your response as JSON using the schema I provided."

Can I customize the industries or target revenue range?
Yes â€” ask your manager or the person who set up the tool. The research prompt is embedded in the HTML file and can be edited in any text editor (Notepad, TextEdit) in about 5 minutes.
